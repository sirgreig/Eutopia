# Eutopía

A mobile remake of the 1981 Intellivision classic **Utopia** — the first god game and city builder.

## About

Eutopía (Greek: εὐτοπία, "good place") brings the strategic depth of Don Daglow's pioneering game to iOS and Android with modern visuals while preserving the original gameplay that influenced SimCity, Civilization, and the entire god game genre.

## Features

- **Original Mode**: Authentic gameplay with full opponent visibility and classic buildings
- **Enhanced Mode**: Fog of war, scouting mechanics, and 6 new building types
- **Single Player**: AI opponent with adjustable difficulty
- **Multiplayer**: Real-time 2-player matches via network

## Tech Stack

- React Native + Expo
- TypeScript
- Zustand (state management)
- react-native-svg (graphics)
- react-native-reanimated (animations)
- WebSocket (multiplayer sync)
